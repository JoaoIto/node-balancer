# Contribuindo para o Node Balancer

Obrigado por considerar contribuir para este projeto!

## Como Contribuir

1.  **Fork o repositório**: Faça um fork deste projeto para a sua conta.
2.  **Crie uma Branch**: `git checkout -b feature/MinhaFeature`
3.  **Faça suas alterações**: Implemente sua feature ou correção de bug.
4.  **Commit**: `git commit -m 'feat: Adiciona nova funcionalidade'`
5.  **Push**: `git push origin feature/MinhaFeature`
6.  **Abra um Pull Request**: Descreva suas alterações e envie para análise.

## Padrões de Código

-   Utilizamos **TypeScript**.
-   Mantenha o código limpo e comentado onde necessário.
-   Certifique-se de que os testes (`npm run ops:demo`) continuam passando.

## Reportando Bugs

Se encontrar um problema, abra uma **Issue** no GitHub com:
-   Descrição do erro.
-   Passos para reproduzir.
-   Logs ou screenshots.

Obrigado!
